Here are some common methods to interact with the DOM:

Select Elements:

document.getElementById("id") – Selects an element by its id.

document.querySelector("css-selector") – Selects the first matching element.

document.querySelectorAll("css-selector") – Selects all matching elements.

Modify Content:

element.textContent = "new text" – Changes the text inside an element.

element.innerHTML = "new HTML" – Changes the HTML inside an element.

Modify Styles:

element.style.property = "value" – Changes the style of an element (e.g., element.style.color = "red").

Add/Remove Elements:

document.createElement("div") – Creates a new element.

parentElement.appendChild(newElement) – Adds a new element to the page.

element.remove() – Removes an element from the page.
